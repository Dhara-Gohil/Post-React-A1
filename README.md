# Task Manager App

A full-stack task management application built using **Next.js**, **Firebase**, **Redux Toolkit**, and **Tailwind CSS**.

## Features

* User authentication with Firebase (signup, login, logout)
* Role-based access (Admin & User)
* Create, update, delete, and assign tasks
* Task status tracking (todo, in-progress, done)
* Real-time updates using Firestore listeners
* Search, filter, and pagination for tasks
* Admin dashboard with user list and task analytics
* Responsive UI with Tailwind + MUI components

## Tech Stack

* Next.js (App Router + API routes)
* Firebase Auth & Firestore
* Redux Toolkit
* React Hook Form
* Tailwind CSS + MUI

## Setup

1. Install dependencies
   `npm install`

2. Add Firebase config in `.env.local`

3. Run development server
   `npm run dev`
